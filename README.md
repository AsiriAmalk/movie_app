# Movie App with Django

### This performs Django CRUD operations
#### You can Create, Read, Update and Delete a movie from the repository
#### Frontend Bootstrap, HTML, JS, CSS

### Creating a virtual environment
#### To create a virtual environment, go to your projectâ€™s directory and run venv. 
#### On macOS and Linux:

`python3 -m venv env`

#### On Windows:

`py -m venv env`

### Activating a virtual environment

#### On macOS and Linux:

`source env/bin/activate`

#### On Windows:

`.\env\Scripts\activate`


#### Leaving the virtual environment
`deactivate`

### Install Dependencies
`pip install -r requirements.txt`

### Run Django
`python manage.oy runserver`
#### You will see

###### Starting development server at `http://127.0.0.1:8000/`
